Para fazer um paragrafo é usado a letra P entre < > < />

Caso queira marcar algum texto para deixar visivelmente destacado,
pode ser utilizado o comando mark entre < > < />

A tag small entre < > < /> Coloca o texto em uma letra de menor tamanho.
A tag big entre < > < /> Coloca o texto em uma letra de maior tamanho.

Para colocar uma palavra ou trecho em Itálico, ou em Negrito, ou até mesmo subrilhando é usado dessa forma:

  • i entre < > < /> para o Itálico
  • strong entre < > < /> para o Negrito
  • u entre < > < /> para o Sublinhado

Caso queira fazer uma lista ordenada devo usar a tag assim: ol entre < > < /> e dentro desta tag também é usado a tag li entre < > < /> para incluir as listas ordenadas.

  1. Assim teremos uma lista ordenada;
  2. Muito funcional;
  3. ótimo para todas as horas que precisar!

Mas caso queira fazer uma lista não ordenada deve usar a tag assim: ul entre < > < /> e dentro desta tag também é usado a tag li entre < > < /> para incluir as listas não ordenadas.

  • Assim teremos uma lista não ordenada;
  • Muito simples e funcional;
  • ótimo para todas as horas que precisar!

Ao longo desse texto inclui a tag hr entre < > para ilustrar a função de uma linha dividindo todos os tópicos, incluindo esse também.

Para fazer uma citação, devo usar blockquote entre < > < />

Dessa forma a citação pode ser feita em meio a tudo o que estiver escrevendo.
Fora da área de marcação, tudo volta ao normal.

Outra tag muito importante para quem quiser simbolizar elementos matematicos, é utilizando sub ou sup entre < > < />
Ficaria assim:
Uma expressão Matematica que tem X2 + Y-3 = 10. Resolva a equação.

Ao final, não menos importante podemos usar a tag font entre < > < /> será utilizado para configurar o tipo, cor e tamanho das letras de um texto de uma página html

A tag del entre < > < /> faz com que seja feita uma linha em cima do texto.

A tag abbr entre < > < /> permite a abreviação de um cabeçalho longo de modo que ele não seja lido por inteiro toda vez que o leitor de tela o encontrar, lendo apenas a abreviação nas demais vezes EXPLICAÇÃO.
